# E-Commerce Sales Analysis (Tableau BI)

## Overview
This project analyzes the Brazilian E-Commerce Public Dataset by Olist to extract meaningful business insights related to sales performance, logistics efficiency, and customer satisfaction.

The project was developed as part of a Data Warehousing & Business Intelligence course, focusing on transforming raw data into actionable insights using data visualization tools.

## Ojectives
- Analyze sales trends and seasonality
- Identify top-performing product categories
- Evaluate logistics and delivery performance
- Understand the impact of delivery speed on customer satisfaction

## Tools & Technologies
- Tableau Desktop
- Tableau Prep Builder
- Data Visualization
- Business Intelligence Techniques

## Dataset
The dataset used in this project is publicly available on Kaggle:
[Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

Note: Due to file size limitations, the dataset is not included in this repository. Please download it from the original source.


## Data Preparation
Data was cleaned and transformed using Tableau Prep, including:
- Handling missing values
- Joining multiple tables
- Creating calculated fields

## Key Insights
**1. Sales Trend**
- Sales show strong growth in early periods but decline toward the later months
- Indicates potential challenges in maintaining long-term revenue growth

**2. Product Performance**
- Top revenue-generating categories include:
    -  Health & Beauty
    - Watches & Gifts
    - Bed, Bath & Table
- Revenue is concentrated in a few categories → potential business risk

**3. Logistics Efficiency**
- Orders are delivered 10–17 days earlier than estimated
- Logistics performance is a key competitive advantage

**4. Customer Satisfaction**
- Negative correlation between delivery delays and review scores
- Faster delivery leads to higher customer satisfaction

## Business Recommendations
- Improve strategies for off-peak sales periods
- Diversify product categories to reduce dependency on top performers
- Maintain strong logistics performance as a core strength
- Identify and resolve delay-prone areas to protect customer satisfaction
